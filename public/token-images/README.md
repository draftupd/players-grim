Put role token images here.

The app tries to match images by role id and role name, for example:

- `savant.png`
- `Savant.png`
- `highpriestess.webp`
- `High Priestess.jpg`
- `High-Priestess.svg`

Supported extensions: `png`, `webp`, `jpg`, `jpeg`, `svg`.

If no image is found for a role, the app uses the current text token style.
